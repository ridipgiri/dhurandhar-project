# Dhurandhar RAG System

A local Retrieval-Augmented Generation system using **Streamlit**, **FAISS**, and **Groq (Llama 3.1)**.

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt

## Setup
1. `pip install -r requirements.txt`
2. `streamlit run streamlit.py` (recommended) or `streamlit run app.py`

## Features
 - Upload PDF or TXT files.
 - Automatic text extraction and chunking.
 - Fast vector search using FAISS.
 - Response generation via Groq Llama 3.1.
